# MyTaskTracker

# 📝 Simple Task Manager (Node.js + Express)

This is a basic web-based Task Manager built with Node.js and Express. It allows users to add and delete tasks using a minimal frontend interface. Tasks are stored temporarily in memory (not saved on restart).

---

## ✅ What’s Done So Far

- [x] Set up Node.js project
- [x] Created a simple `server.js` using Express
- [x] Built a static front end (HTML, CSS, JS)
- [x] Implemented basic task operations:
  - Add task
  - View tasks
  - Delete task
- [x] Tested locally at `http://localhost:3000`

---

## 🛠️ How to Run It

```bash
# Step into the project directory
cd "C:\Users\haala\New folder\python\tasktracker"

# Install dependencies
npm install

# Start the server
node server.js
Then open your browser at http://localhost:3000
